Images

Images are used in every web page and it's very important to make them responsive.

Responsive images

img-responsive - Makes the image fit the width of container; it's height will get adjusted by keeping the aspect ratio same, if you want to change the aspect ratio, you will have to crop the image

Note: The second container has a width of 50%, see how the image is changing depending on the size of the container. 


<img
    class="img-responsive"
    src="https://picsum.photos/1000/400"
    alt="responsive image"
/>
