Toast

Toasts are lightweight notifications designed to mimic the push notifications that have been popularized by mobile and desktop operating systems.

toast - The main class; to be given to the wrapper div element

toast-close - Optional; to be given to the close button in case the toast requires one

toast-link - Optional, to be given to the anchor tag which requires an action
