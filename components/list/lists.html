List

Lists are used to display items in an orderly manner.

Ordered list

ordered-list - Main class; to be given to the ol element which has the list items

list-title - To be given if there's a heading for the list items

    Fun Games
  1. Tumblin dice
  2. Chameleon
  3. Ooga Booga
  4. Fastrack
  5. Geistz

<ol class="ordered-list">
    <div class="list-title">Fun Games</div>
    <div class="divider-black"></div>
    <li>Tumblin dice</li>
    <li>Chameleon</li>
    <li>Ooga Booga</li>
    <li>Fastrack</li>
    <li>Geistz</li>
</ol>

Unordered list

unordered-list - Main class; to be given to the ul element which has the list items

    Group Games (4+)
  • Taboo
  • Tsuro
  • Bang - The dice game
  • Coup
  • Hail Hydra

<ul class="unordered-list">
    <div class="list-title">Group Games</div>
    <div class="divider-black"></div>
    <li>Taboo</li>
    <li>Tsuro</li>
    <li>Bang - The dice game</li>
    <li>Coup</li>
    <li>Hail Hydra</li>
</ul>
