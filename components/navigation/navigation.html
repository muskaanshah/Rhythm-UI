Navigation menu

A navigation bar (or navigation system) is a section of a graphical user interface intended to aid visitors in accessing information. This helps the user to navigate between the various pages

nav - The main class; to be given to the wrapper div / ul element

nav-item - To be given to each of the child p/li/a tags which contains the link

nav-logo - Optional; can give text, brand's logo in this class

nav-logo-img - Optional; to be used if an image is being given for the logo; if text is given, this can be omitted

  • Home
  • About
  • Services
  • Contact us
