Avatar

Avatar icons are used to display a small profile picture for the user who logs in.

Picture avatar

avatar - The main class; has to be used when the avatar has to be displayed; can be styled with other classes to give other effects; has to be given to the img class

avatar-lg - Optional; to be used when the avatar needs to be larger than usual

alert-sm - Optional; to be used when the avatar needs to be smaller than usual

Note: The border radius is changing by using the borderradius property as mentioned in the Utilities section

Initials avatar

avatar-default - for default size; has to be used when the avatar has to be displayed in initials format; can be styled with other classes to give other effects; can be given on a span or a div class

avatar-default-lg - to be used for a larger size size

avatar-default-sm - to be used for a smaller size

Note: The colour styles are used for styling the avatar
