Utilities

Rhythm has a variety of utility classes which helps you customise your page in the exact way that you want it to be.

Padding

p - this specifies the padding on all 4 sides of the element

px - this specifies the padding on the x-axis (left and right only)

py - this specifies the padding on the y-axis (top and bottom only)

pt - this specifies the padding only on the top

pb - this specifies the padding only on the bottom

pr - this specifies the padding only on the right

pl - this specifies the padding only on the left

Margin

m - this specifies the margin on all 4 sides of the element

mx - this specifies the margin on the x-axis (left and right only)

my - this specifies the margin on the y-axis (top and bottom only)

mt - this specifies the margin only on the top

mb - this specifies the margin only on the bottom

mr - this specifies the margin only on the right

ml - this specifies the margin only on the left

Values

0 - value of this is 0rem

0-5 - value of this is 0.5rem

1 - value of this is 1rem

1-5 - value of this is 1.5rem

2 - value of this is 2rem

2-5 - value of this is 2.5rem

3 - value of this is 3rem

The user can add any of the styles mentioned in the padding and margin section with any value which suits their need for more customisation.

Note: Apart from the above mentioned tags, we also have px-auto and mx-auto which can be used to set the padding and margin on both sides respectively. 


/* Padding and margin to be used like this along with the values */
<div class="pr-1 py-2-5 mb-0-5"></div>

Border radius

borderradius-0 - border radius is not applied
borderradius-0-5 - border radius of 0.5rem
borderradius-1 - border radius of 1rem
borderradius-1-25 - border radius of 1.25rem
borderradius-1-5 - border radius of 1.5rem
borderradius-2 - border radius of 2rem
border radius of 50% 

<div class="borderradius-0">border radius is not applied</div>
<div class="borderradius-0-5">border radius of 0.5rem applied</div>
<div class="borderradius-1">border radius of 1rem applied</div>
<div class="borderradius-1-25">border radius of 1.25rem applied</div>
<div class="borderradius-1-5">border radius of 1.5rem applied</div>
<div class="borderradius-2">border radius of 2rem applied</div>
<div class="borderradius-full">border radius of 50%</div> /* gives circle when square element */

Box shadow

boxshadow-light - box shadow is of a very light density

boxshadow-medium - box shadow is of a medium density

boxshadow-dark - box shadow is of a dark density

boxshadow-emboss - box shadow is applied only on two sides and gives an embossed effect

Light shadow
Medium shadow
Dark shadow
Embossed shadow

<div class="boxshadow-light bg-success-light">Light shadow</div>
<div class="boxshadow-medium bg-danger-light">Medium shadow</div>
<div class="boxshadow-dark bg-warning-light">Dark shadow</div>
<div class="boxshadow-emboss bg-secondary">Embossed shadow</div>

Other misc

centered - centers the contents according to x-axis and y-axis - has to be mentioned on the parent container

divider-black - gives a small thin line in between 


<div class="centered"><div class="divider-black"></div></div>
