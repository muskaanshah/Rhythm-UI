Buttons

Buttons are used in almost every website in forms, toolbars, homepages, dialog boxes. If the user has to do an action or submit data to the server, a button is used to initiate the action.

General buttons

btn - The main class; has to be used when the user wants a button; to be given for all variation of buttons; can be styled with other classes to give other effects; has to be given to the button / anchor tag

btn-disabled - Optional; to be used when the user shoudn't be allowed to click on a button

btn-lg - Optional; to be used when a larger button is required

btn-sm - Optional; to be used when a smaller button is required

Outlined buttons

btn-primary-outline - to be used when user wants a button with the primary shade as the border

btn-accent-outline - to be used when user wants a button with the accent shade as the border

Note: For the primary shade and accent shade alone there is an inbuilt class, for the other shades, you can used the border and color property to style the button as per your need
