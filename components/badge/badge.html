Badge

Badges are small elements which are used to indicate something. They can notify you that there are new or unread messages or notifications, or indicate the status of a user.

Number badge

badge-container - Has to be given to the wrapper class which is wrapping the icon and the badge

badge - Has to be given to the child div containing the number

badge-number - Has to be given to the child div containing the number; this indicates that the badge type is a number

notifications
17

</div>

    </div>
</div>

Status badge

badge-container - Has to be given to the wrapper class which is wrapping the avatar and the badge

badge - Has to be given to the child div which shows the status

badge-status - Has to be given to the child div to state that a status badge is used

badge-online - Has to be given to the child div to show online status of the user

badge-busy - Has to be given to the child div to show busy status of the user

badge-offline - Has to be given to the child div to show offline status of the user; this is the default status if no status class is applied
